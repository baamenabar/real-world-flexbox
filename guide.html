Formularios y sus elementos

Los elementos de un form, en general, son por defecto display:inline; y display:inline-block; por lo que es mejor meterlos dentro de un bloque para que no quede una majamama de elementos.

Personalmente prefiero las listas y en los estilos defino que los li dentro de un form se comportan parecido a un div.

  • Por favor ingrese correctamente sus datos.

  • Este es un <li> con clase .transparente por lo que no tiene fondo. También se pueden agregar las clases .pegado para un margen pequeño hacia arriba y .sin-margen para no tener margen hacia arriba.

  • Para los input[checkbox] y input[radio] agregaremos la clase .dentro y envolveremos los input en su label de la siguiente manera:

    <li class="dentro">
<label for="nombre6">Etiqueta para checkbox 
<input type="checkbox" name="nombre6" id="nombre6"></label>
</li>

  • Para múltiples input[checkbox] y input[radio] podemos una serie de <label> dentro del <li> de la siguiente manera:

    <li class="ancha dentro">
<label><input type="radio" name="grupoIn"> etiqueta 1</input></label>
<label><input type="radio" name="grupoIn"> etiqueta 2</input></label>
<label><input type="radio" name="grupoIn"> etiqueta 3</input></label>
</li>

  • Ojo que un button debe tener type="button" si no se quiere que envíe automáticamente el formulario.

  • A veces es útil tener estilos para que link parezca botón a.button o un botón pequeño small link a.button.small
